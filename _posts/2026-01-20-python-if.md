## 조건문 (if)

- `if`-`elif`-`else` 문을 사용한다
- `and`, `or`, `not`을 사용한다
- `in`, `not in`을 야무지게 사용하도록 하자

```python
a = [1, 2, 3, 4, 5]
b = 7
if b not in a:
    print(f'{b} is not in a ({a})')
else:
    print(f'{b} is in a ({a})')
```
